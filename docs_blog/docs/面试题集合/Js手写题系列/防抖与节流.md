```markdown
# 防抖：连续触发某个事件时，只有当一定时间内没有再次触发，才会执行函数。
## 常用场景：搜索框
```

```js
function debounce(fn,delay){
  let timer
  return function (...args){
    clearTimeout(timer)
    timer = setTimeout(()=>{
      fn.call(this,args)
    },delay)
  }
}
```

```markdown
# 节流：无论点击多少次，规定时间内只触发一次。
## 常用于快速点击，如微博抖音点赞
```

```js
function throttle(fn,delay){
  let timer 
  return function(...args){
    if(!timer){
      timer = setTimeout(()=>{
        fn.call(this,args)
        timer = null
      },delay)
    }
  }
}
```
